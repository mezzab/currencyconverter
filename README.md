# CurrencyConverter

A currency converter web application.

## Development

### Installation

Install dependencies
   ```
   $ yarn install
   ```
   
### Development build   
Run the web application with hot reload
   ```
$ yarn start-web
   ```

### Running linting process
This ensure we follow the styleguide.
   ```
$ yarn run lint
   ```
